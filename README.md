# isMedia NFT Contracts

ERC1155, ERC721, and marketplace V1 contracts for isMedia Ethereum NFTs

## Usage

[Hardhat](https://hardhat.org/) is used for contract development and deploy.

Install packages
```
npm install
```

Compile contracts
```
npm run compile
```

Run tests
```
npm run test
```
